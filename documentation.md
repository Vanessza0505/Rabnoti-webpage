# Rabnoti-webpage
I made this website for a school competition.
Several of us worked on this competition. I was mostly responsible for the creativity and the website in the project.

The projekt:
My team and I designed an escape room game that takes place at our school.
It works in 2 player mode, one player controls the game itself while the other helps from the website, the two players have to work together to get out.
The whole project includes backend and frontend and the game itself is written in godot.
I solved the frontend with tailwind css and react.
